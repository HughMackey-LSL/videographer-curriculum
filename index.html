About This Course

This course is designed to build practical videography skills from a near-beginner level. The emphasis is on independent, reliable field production: operating a camera for interviews and events, collecting useful coverage, setting up basic lighting with a minimal kit, and recording quality sound.

Work through each module in order. Complete the lessons, answer the knowledge checks along the way, and take the module quiz before moving on. Assignments are practical exercises to reinforce what you have learned.

How to use this course

Each lesson takes 10–20 minutes. Read actively — try to connect each concept to a camera or situation you have encountered. The module quizzes and assignments are where the learning actually sticks.

Readiness Criteria

You are ready to operate more independently when you can consistently do the following. Use this list as an honest self-assessment at the end of the course.

  • Set camera exposure, focus, white balance, resolution, and frame rate correctly.
  • Capture clean dialogue audio and monitor it during recording.
  • Set up basic interview lighting with a small kit.
  • Frame subjects professionally.
  • Capture stable, usable B-roll and event coverage.
  • Think in terms of shots, sequences, and edit needs.
  • Cover key event moments without constant prompting.
  • Organize and hand off media correctly.
  • Notice technical problems while there is still time to fix them.
  • Explain setup choices clearly.
  • Produce footage that an editor can use without major rescue work.
